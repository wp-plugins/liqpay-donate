Liqpay Donate - plugin for to get donations and accept payments.

== Description ==

Another small plugin for donations. This plugin allows you get donations and accept payments from people using <a href="https://www.liqpay.com/">LiqPAY</a> system. 
LiqPAY is the swiftly developing payment system in Europe. 
<a href="http://www.icom.kiev.ua/index.php?page=liqpay">Visit widget page</a> for more information.

== Installation ==

To set the plugin up, you need to:

1. Upload the liqpay directory to the /wp-content/plugins/  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag widgets on a side bar
